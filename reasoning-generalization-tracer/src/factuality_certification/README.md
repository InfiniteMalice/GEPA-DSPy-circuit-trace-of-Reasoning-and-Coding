# GeoCert-Inspired Factuality Certification, Observability, and Over-Refusal Guard

Constraint-based certification layer relative to available evidence and context (not a formal truth proof).

Modes: `off`, `shadow`, `advisory`, `gated`, `training`.

This layer distinguishes refusal vs abstention vs scoped/qualified answers and preserves positive-only
thought-trace rewards (no negative scoring on hidden reasoning traces).
